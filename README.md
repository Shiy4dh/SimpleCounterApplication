# CounterApp

A simple Java console application that counts numbers. This project demonstrates basic Java programming, including loops, input handling, and simple logic.

## Features

- Increment and decrement a counter
- Display the current value
- Simple console-based interface
- Easily extendable for additional functionality

## Prerequisites

- Java Development Kit (JDK) 25 or higher
- JavaFX SDK (if not bundled with your JDK)
- IntelliJ IDEA or any Java IDE (optional)

## How to Run

### Using IntelliJ IDEA (Recommended)

1. Open IntelliJ IDEA and select **Open** → navigate to your project folder.
2. Make sure the JavaFX library is configured:
   - Go to **File → Project Structure → Libraries → + → Java**.
   - Select your JavaFX SDK folder (e.g., `javafx-sdk-XX/lib`) and click **OK**.
3. Open the `CounterApp.java` file containing the `main` method.
4. Click the **Run** button (green triangle) at the top-right corner of IntelliJ.
5. The JavaFX window with your counter should appear.
